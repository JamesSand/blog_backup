Computer Organization

原码：

计算原码只要在转化的二进制数前面加上相应的符号位就行了。原码可以表示任何的数字。0 只有一个原码，因为它算正数，符号位为 ‘0’

数值位就是真值的绝对值，符号位为“0”时表示正数，符号位为“1”时表示负数，原码又称带符号的绝对值。

反码：

任何反码表示的数都有一个原码表示。

对于负数，将原码各位取反，符号位不变。

对于正数，反码 = 原码

补码：

对于负数，补码 = 反码 + 1

对于正数， 补码 = 原码

IEEE 单精度浮点数

注意 Exp 位 全0 和 全1 有特殊的表示意义，在计算最大值和最小值的时候需注意

字符编码

ASCII 码使用一个字节( 8 bits )编码，只能编码英文。

对 UTF-8 来说，其他语言的字符（包括中日韩文字、东南亚文字、中东文字等）包含了大部分常用字，使用3字节编码。

矢量图

所谓矢量图，就是使用直线和曲线来描述的图形，构成这些图形的元素是一些点、线、矩形、多边形、圆和弧线等，它们都是通过数学公式计算获得的，具有编辑后不失真的特点。

海明码

假设数据位为 k 校验位为 r 校验位总共有 2r2^r2r 种表示，其中一种表示正确，剩下每一种指出一个位的错误，总共有 k + r 位，所以有 2r−1≥k+r2^r-1 \geq k + r2r−1≥k+r

SRAM

ram_ce_n：片使能，0 表示 SRAM 使用，1 表示省电

总线协议

CPU 需要告诉内存，什么时候需要进行读写操作，用 valid 信号表示。

内存要有一个信号告诉 CPU 是否操作完成，用 ready 信号表示。

master 用来表示 CPU 端，一些不太显然的信号如下：

  • we_o：master 想读还是想写
  • be_o：字节势能，实现单字节读写

Wishbone 可以分为 master 和 slave 两部分，master 对应 CPU，slave 对应 SRAM

URAT 协议

全双工信道

  • rx / rxd recieve 接收
  • tx / txd transmit 发送

我们通常把 UART 协议中 1s 时间间隔的个数记为波特率（Baud Rate）。常见的波特率有 9600 和 115200。

串口控制器

问题在于，时钟频率和波特率并不相同，往往时钟频率更高。接收串口上一个 bit 可能需要多个时钟周期。

Modern Biology

高能磷酸化合物

高能磷酸键指的是能量大于 5 kcal / mol

PCr 磷酸肌酸

GTP：和 ATP 碱基不一样

高能磷酸化合物将能量先传给 ATP，再由 ATP 去真正供能

ATP ，ADP 和 AMP（ATP 水解不完全的产物） 的比值可以作为调控因子

细胞呼吸

有氧呼吸：糖酵解，三羧酸循环，电子传递与 ATP 形成

糖酵解发生在细胞质中，产物是丙酮酸

光合作用

光合单位 = 天线色素系统 + 反应中心色素

  • 天线色素系统：收集光能，大部分
  • 反应中心色素：少部分既能捕获光能，又能转化光能的色素。

光反应：

光能的吸收，转换成化学能
